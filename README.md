StickyNotes
--
**Dependencies**
- RPM: qtbase6-common qt6pas
- DEB: libqt6core6 libqt6gui6 libqt6widgets6 libqt6dbus6 libqt6pas6

Simple, colorful notes for everyday work. The size of the note changes the lower-right corner. Individual note colors, and font can be customized. It is possible to import and export all notes to a file. After startup, it shows a yellow icon in the system tray and is controlled from there. One-click on the icon to show/hide all notes.  
  
Простые, красочные заметки для повседневной работы. Размер заметки изменяется за правый нижний угол. Настраиваются индивидуальные цвета заметок и шрифт. Есть возможность выполнить импорт и экспорт всех заметок в файл. После запуска показывает желтую иконку в системном трее и управляется оттуда. Одиночный клик по иконке отображает/скрывает все заметки.
