Sticky Notes
--
Simple, colorful stycky notes for everyday work. The size of the note changes  the lower-right corner. Individual note colors, font color, and font style can be customized. It is possible to import and export all notes to a file. After startup, it shows a yellow icon in the system tray and is controlled from there.

Made in Mageia-8, additionally tested in Linux Mint-20.

Простые, красочные заметки для повседневной работы. Размер заметки изменяется за правый нижний угол. Настраиваются индивидуальные цвета заметок, цвет и стиль шрифта. Есть возможность выполнить импорт и экспорт всех заметок в файл. После запуска показывает желтую иконку в системном трее и управляется оттуда.

Сделано в Mageia-8, дополнительно протестировано в Linux Mint-20.
